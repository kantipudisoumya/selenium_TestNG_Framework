\# Selenium TestNG Automation Framework



\## Overview

This is a Selenium automation framework built using Java, TestNG, and Maven.  

It follows Page Object Model (POM) and Page Factory design patterns.



\## Tech Stack

\- Java

\- Selenium WebDriver

\- TestNG

\- Maven



\## Framework Design

\- Page Object Model (POM)

\- Page Factory (@FindBy)

\- Driver Factory pattern

\- TestNG XML-based execution



\## 📂 Project Structure

src/

&#x20;├── main/java

&#x20;│   ├── factory

&#x20;│   ├── pages

&#x20;│   ├── utilities

&#x20;├── test/java

&#x20;│   ├── tests



\## How to Run

Run via testng.xml





Features:

Cross-browser execution (Chrome, Firefox)

Parameterized execution using TestNG

Clean driver initialization using Factory pattern

Reusable page objects
